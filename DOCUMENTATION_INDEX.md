## 📋 Quick Checklist

### Before You Start
- [ ] Read GETTING_STARTED.md
- [ ] Have Node.js v18+ installed
- [ ] Redis installed and running
- [ ] Supabase account created
- [ ] Anthropic API key obtained

### During Setup
- [ ] npm install completes
- [ ] .env file created with all variables
- [ ] Database schema imported to Supabase
- [ ] npm run dev starts both servers

### After Running
- [ ] Can access frontend on 5173
- [ ] Can access backend on 3001
- [ ] Can register/login
- [ ] Can view campaigns page
- [ ] Network tab shows API calls with auth headers

### If Issues Arise
- [ ] Check TROUBLESHOOTING.md
- [ ] Verify .env variables
- [ ] Check Redis is running
- [ ] Check database schema imported
- [ ] Look at browser console errors
- [ ] Look at server logs
